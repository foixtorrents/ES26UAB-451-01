# Requirement

### ID:
RF-3-02

### Type/Category:
Funcional

### Version:
1.0

### Title:
Càlcul del cost d'enviament

### Description:
El sistema ha de calcular automàticament el cost d’enviament a partir de la distància entre el cuiner i el punt de recollida. 
En el cas que el client reculli la comanda personalment, el cost serà zero.
