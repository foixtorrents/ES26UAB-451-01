# Requirement

### ID:
RNF-A-08

### Type/Category:
No Funcional / Eficiència (Rendiment)

### Version:
1.0

### Title:
Latència màxima de resposta del sistema

### Description:
El sistema ha de garantir una latència de resposta extremadament baixa, establint un temps màxim de 5 mil·lisegons per a completar el processament de qualsevol petició realitzada per l'usuari.
